# Saffron Landing Page

An elegant, simple landing page for Saffron - Your AI Trading Co-Pilot.

## Features

- Clean, modern design with saffron color theme
- Hero section with waitlist CTA
- Feature showcase explaining what Saffron does
- Mobile-first responsive design
- Social media integration

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Customization

- Update the waitlist form link in `src/App.js` (line 17)
- Modify colors in `src/App.css` to adjust the saffron theme
- Add your own content and features as needed

## Tech Stack

- React 18
- Create React App
- CSS3 with custom styling
- Google Fonts (Inter)
